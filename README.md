# 捷途车机口令工具

专为捷途车主设计的**车机&ADB动态口令计算器**，支持多种车型和版本，基于当前时间自动生成有效口令。


## 🚗 功能说明
1. **工程模式口令**：用于进入工程模式（应用中心→蓝牙电话，输入上方口令）。  
2. **ADB权限口令**：用于工程模式加密设置（进入工程模式→加密设置，输入上方口令）。  
3. **自动更新**：口令根据算法类型自动更新（每小时/每天/固定口令）。  
4. **多车型支持**：支持捷途旅行者、自由者、大圣等多种车型。  


## 📱 适用车型

### 捷途旅行者/山海T2
- **00.08及以下**：基于序列号计算ADB口令
- **4.06及以下**：固定工程模式口令
- **4.07以上**：动态口令（每小时更新）
- **其他**：基于序列号计算ADB口令

### 自由者/山海T1
- **11.01.04及以上**：动态口令（每小时更新）

### 捷途大圣
- **固定口令**：固定工程模式口令

### 其他车型
- 瑞虎8/Pro、风云A9、山海L7、山海L9、X70plus等

## 💻 使用方法
1. 在线使用：访问演示地址直接使用
2. 本地使用：下载本仓库的所有文件，双击 `index.html` 即可在浏览器中打开使用（无需服务器）


## ⚠️ 注意事项
- 动态口令基于**当前时间**计算，若设备时间不准会导致口令错误，请确保系统时间正确。  
- 不同车型和版本使用不同的算法，请选择正确的车型和版本。  
- 底部说明中的车型适配以官方信息为准，如有变动请自行核对。  


## 🛠 技术栈

- JavaScript：时间计算+动态口令生成  
- 模块化设计：算法独立封装，便于维护和扩展

# 捷途车机动态密码生成工具

专为捷途山海T2/旅行者车主设计的**车机&ADB动态密码计算器**，基于当前时间自动生成有效密码（每小时更新）。


## 🚗 功能说明
1. **车机动态密码**：用于蓝牙电话拨号（打开应用中心→蓝牙电话，输入上方密码）。  
2. **ADB权限密码**：用于工程模式加密设置（进入工程模式→加密设置，输入上方密码）。  
3. **自动更新**：每分钟刷新一次，确保密码时效性。  


## 📱 适用车型
- ✅ 2024款捷途山海T2  
- ✅ 2023/2024款捷途旅行者  
- ❌ 银龙之翼/2025款车型不适用

## 💻 使用方法
1. 下载本仓库的所有文件（`index.html`、`style.css`、`app.js`）。  
2. 直接双击 `index.html` 即可在浏览器中打开使用（无需服务器）。  


## ⚠️ 注意事项
- 密码基于**当前时间**计算，若设备时间不准会导致密码错误，请确保系统时间正确。  
- 底部说明中的车型适配以官方信息为准，如有变动请自行核对。  


## 🛠 技术栈
- HTML5：页面结构  
- CSS3：响应式布局+样式美化  
- JavaScript：时间计算+动态密码生成
